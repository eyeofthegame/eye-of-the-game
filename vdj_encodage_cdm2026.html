Instructions

Module Statistiques — Encodage des buts

Ce module permet d'encoder les détails des buts selon la méthode Vision du Jeu (9 couches d'analyse).

Procédure :

  1. Sélectionnez un match dans la liste
  2. Entrez le score si ce n'est pas déjà fait
  3. Pour chaque but, remplissez : minute, buteur, assist
  4. Encodez les 9 couches VDJ dans l'ordre : Tactique 1, Origine 1, Origine 2, Origine 3, Lieu, Action, Tactique 2, Nature du but
  5. Choisissez la finition (But, But marqué, etc.)
  6. Validez l'encodage

Rappels importants :

  • Les buts en arrêts de jeu : minute 45 (1ère MT) ou 90 (2e MT)
  • TOUJOURS valider les stats, même pour un 0-0
  • Les données sont sauvegardées dans le navigateur (localStorage)
  • Pensez à exporter régulièrement en JSON pour sauvegarder
